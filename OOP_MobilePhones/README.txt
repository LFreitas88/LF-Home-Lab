Description:
------------
This project demonstrates Object-Oriented Programming in Python
by defining a class named "Mobile" that represents a mobile phone.

Files:
------
1. mobile.py  - Defines the Mobile class and its methods.
2. main.py    - Imports the Mobile class and creates 4 phone objects.
3. README.txt - Explains the purpose of this project.

Features:
---------
- Demonstrates class creation, attributes, and methods.
- Uses "self" to access instance attributes.
- Simulates real-world phone actions like calling and gaming.
- Shows how to import a class from another file.

How to Run:
-----------
1. Place both files (mobile.py and main.py) in the same folder.
2. Open a terminal or command prompt in this folder.
3. Run the command:

   python3 main.py
